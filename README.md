 

# 📊 Student Grade Management System

## 📌 Overview

The **Student Grade Management System** is a simple Python console application that allows users to manage student records. It enables adding student names with their grades, viewing all records, calculating the average grade, and finding the highest and lowest scores.

This project is designed for beginners to understand **Python basics such as dictionaries, functions, loops, and conditional statements**.

---

## 🚀 Features

* ➕ Add student name and grade
* 📋 View all student records
* 📊 Calculate average grade
* 🏆 Find highest and lowest grades
* ❌ Exit the program safely

---

## 🛠️ Technologies Used

* Python 3.x
